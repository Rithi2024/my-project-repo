# Frontend (Flutter)

## Setup
```bash
flutter pub get
```

## Run (Android Emulator)
```bash
flutter run
```

## API Base URL
```
http://10.0.2.2:3000
```

## Features
- JWT Authentication (Login / Logout)
- Category CRUD with debounced search (Khmer supported)
- Product CRUD with pagination, infinite scroll, sorting, filtering
- Images loaded from backend and cached locally on device
